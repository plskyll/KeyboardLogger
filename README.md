# Інструкції з використання логера
Gростий Логер, який відстежує дії користувача в програмі та
зберігає ці логи у текстовому документі. Цей Логер використовується для
запису різних дій користувача під час використання програми.
## Встановлення та запуск

### 1. Клонування репозиторію
Спершу завантажте проект на свій комп'ютер. Відкрийте термінал і виконайте команду:

```bash
    git clone https://github.com/plskyll/KeyboardLogger.git
```

### 2. Налаштування оточення (Рекомендовано)
Бажано створити віртуальне оточення, щоб не засмічувати систему:
```bash
    python -m venv venv
```
Далі активувати:
```bash
    # Для Windows:
    venv\Scripts\activate
```
```bash
    # Для macOS/Linux:
    source venv/bin/activate
```

### 3. Встановлення залежностей
Встановіть всі необхідні бібліотеки однією командою:
```bash
    pip install -r requirements.txt
```

### 4. Запуск скрипту
Запустіть логер командою:
```bash
    python logger.py
```

### 5. Зупинка програми
Для зупинки програми натисніть клавішу Esc на клавіатурі, або зупиніть виконання скрипта за допомогою інтерпретатора.

### Використання
1. Після запуску програма працює у фоновому режимі.
2. Виконуйте будь-які дії мишею або клавіатурою.
3. Щоб зупинити запис і вийти з програми, натисніть клавішу ESC.
4. Результати будуть збережені у файлі activity_log.txt.

### Приклад виводу (activity_log.txt)
```
2025-11-17 22:26:23 - INFO - МИША: Натиснуто Button.left на позиції (1020, 557)
2025-11-17 22:26:24 - INFO - КЛАВІАТУРА: Натиснуто клавішу '1'
2025-11-17 22:26:25 - INFO - КЛАВІАТУРА: Натиснуто спец. клавішу Key.enter
2025-11-17 22:26:30 - INFO - ПРОГРАМА: Користувач ініціював зупинку (ESC)
```
